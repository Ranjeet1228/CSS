# Cursor Landing Page (Tailwind CSS Clone)

## Project Overview

This project is a **frontend recreation of the Cursor website landing page** using **HTML and Tailwind CSS**.
The goal of this project was to practice **responsive layouts, grid systems, flexbox, and Tailwind utility classes**.

---

# Fonts Used

The project uses **system fonts via Tailwind CSS default configuration**.

Typical stack:

```
font-family: ui-sans-serif, system-ui, sans-serif;
```

This provides good performance and compatibility across devices.

---

# Colors Used

The design uses **Tailwind CSS color utilities**:

| Color               | Usage                    |
| ------------------- | ------------------------ |
| `stone-950`         | Dark background          |
| `stone-900`         | Section backgrounds      |
| `gray-900`          | Card backgrounds         |
| `stone-800`         | Content blocks           |
| `gray-400`          | Secondary text           |
| `red-600 / red-700` | Call-to-action links     |
| `white`             | Primary text and buttons |

These colors match the **dark UI style of the Cursor website**.
